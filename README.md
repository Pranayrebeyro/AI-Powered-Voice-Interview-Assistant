# 🎙️ AI-Powered Voice Interview Assistant

An AI-driven interview preparation platform that conducts voice-based mock interviews, analyzes candidate responses, and provides intelligent feedback to improve communication and interview performance.

## 🚀 Features

- Voice-based interview interaction
- AI-generated interview questions
- Speech-to-text conversion
- Real-time response analysis
- Personalized feedback and scoring
- Communication and confidence assessment
- Simple and user-friendly interface

## 🛠️ Tech Stack

### Frontend

- CSS

### Backend

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Pranayrebeyro/AI-Powered-Voice-Interview-Assistant.git
```

### Navigate to Project

```bash
cd AI-Powered-Voice-Interview-Assistant
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Backend

```bash
python app.py
```

### Open Frontend

Open `index.html` in your browser.

## 💡 How It Works

1. User starts a mock interview.
2. AI generates interview questions.
3. Candidate answers using voice.
4. Speech is converted to text.
5. Gemini AI evaluates the response.
6. Detailed feedback and suggestions are displayed.
